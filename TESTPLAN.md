Testplan der Antismoke App: 

Dieses Dokument beschreibt die durchgeführten funktionalen Tests der mobilen App im Rahmen der Modulprüfung:

Testfälle und Ergebnisse:
- 1: Regristierung => E-Mail und Passwort sind korrekt eingegeben => Benutzer wird angelegt => Login erfolgt erfolgreich
- 2: Regristierung ungültig => Passwort zu kurz oder E-mail ungültig => Fehlermeldung wird angezeigt => Erfolgreich
- 3: Login => Korrekte Credentials => App öffnet den Hauptscreen => Erfolgreich
- 4: Login fehlerhaft => falsches Passwort => Fehlermeldung => Erfolgreich
- 5: Quit Date setzten => Datum über DatePicker auswählen => Datum wird angezeigt und die Tage neu berechnet => Erfolgreich
- 6: Smoking Porifle ändern => Kosten und Zigarettenmenge anpassen => Ersparniss und Statistik werden aktualiseirt => Erfolgreich
- 7: Slip Days erhöhen => Andere Zahl eintippen zum Beispiel '3,4,5' => Ersparnis sinkt entsprechend => Erfolgreich
- 8: Logout => Logout Button drücken => Benutzer wird abgemeldet => Erfolgreich
- 9: UI Test => Tastatur verdeckte die Eingabefelder => Felder wieder sichtbar => Fehler korrigiert
- 10: App neu starten => App öffnen => Statistiken werden neu berechnet => Erfolgreich

Fazit:
Alle wesentlichen Funktionen wurden getestet. Die App funktioniert wie erwartet und berechnet korrekt. Notwendige Anpassungen wurden vorgenommen.
